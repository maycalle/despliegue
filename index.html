Desarrollo de videojuegos con Phaser

Esta web es un curso sobre desarrollo de videojuegos utilizando la librería Phaser de JavaScript. Abordaremos primero el software necesario para poder desarrollar videojuegos con esta librería, y después iremos viendo poco a poco conceptos básicos, como dibujo de imágenes, textos, reproducción de audio, paso de pantallas, y otros más avanzados como sistemas de cámaras o partículas

El uso de estos materiales está sujeto a una licencia Creative Commons CC BY-NC. Las imágenes necesarias, salvo que se indique expresamente lo contrario, se han tomado de la web oficial de Phaser, en sus respectivos tutoriales.
