Despliegue de Aplicaciones Web

Esta web recoge los contenidos del módulo de Despliegue de Aplicaciones Web de 2º curso del CFGS DAW impartido en el IES San Vicente.

En este módulo se tratará, por un lado, la configuración de servidores web empleando el framework Node.js, para posteriormente ver cómo desplegar este tipo de servidores y aplicaciones en un entorno real, junto con otros servidores como Nginx, y añadiendo módulos de funcionalidad adicional, como gestores de bases de datos.
